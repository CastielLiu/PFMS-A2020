bool Ranger::setAngularResolution ( unsigned int  angular_resolution)
virtual

Member function sets Angular Resolution of ranger in degrees.

Parameters
angular_resolution
Returns
True if param set successfully, otherwise false

Sets angular resolution according to Ranger's SensingMethod type.

Note
for POINT type, 10 and 30 are the only acceptable values.
for CONE type, angular velocity will be equal to field of view.
All other values will return false.

Implements RangerInterface.
